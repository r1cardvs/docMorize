Stack Usada

Axios

O Axios é uma biblioteca JavaScript que facilita a realização de requisições HTTP assíncronas em navegadores e em ambientes Node.js, oferecendo uma interface simplificada para trabalhar com dados em aplicações web. Saiba mais

Bootstrap

Bootstrap é um framework front-end de código aberto que fornece um conjunto de ferramentas e estilos pré-projetados para facilitar o desenvolvimento de interfaces web responsivas e esteticamente agradáveis. Ele inclui uma variedade de componentes, como grids, botões, formulários e modais, simplificando o processo de criação de layouts consistentes e visualmente atrativos. Saiba mais

CSS

CSS (Cascading Style Sheets) é uma linguagem de estilo que complementa o HTML, permitindo a definição de aparência e formatação, como cores, fontes e layout, proporcionando um design visualmente atraente e consistente às páginas web. Saiba mais

HTML

HTML (HyperText Markup Language) é uma linguagem de marcação utilizada na construção de páginas web, definindo a estrutura e organização do conteúdo, como textos, imagens e links. Saiba mais

Figma

O Figma é uma plataforma de design colaborativo baseada em nuvem, permitindo que equipes trabalhem simultaneamente na criação de interfaces de usuário, protótipos e designs interativos. Oferece recursos de edição em tempo real, facilita a colaboração remota, e permite a criação de designs responsivos e consistentes, contribuindo para um fluxo de trabalho eficiente no desenvolvimento de produtos digitais. Saiba mais

Flask

O Flask é um framework leve para desenvolvimento web em Python. Ele facilita a criação de aplicativos web, seguindo o paradigma de arquitetura MVC (Model-View-Controller). Com uma estrutura simples, o Flask permite construir rapidamente aplicações web escaláveis e flexíveis, sendo amplamente utilizado para desenvolvimento ágil e prototipagem. Suporta extensões para funcionalidades adicionais e é conhecido por sua simplicidade e facilidade de aprendizado. Saiba mais

Git

O Git é um sistema de controle de versão distribuído amplamente utilizado no desenvolvimento de software. Ele permite que desenvolvedores controlem e registrem alterações no código-fonte de maneira eficiente, rastreiem o histórico de versões e facilitem a colaboração entre equipes, possibilitando a ramificação e mesclagem de código de maneira eficaz. Saiba mais

GitHub

O GitHub é uma plataforma de hospedagem e colaboração para controle de versão de código-fonte usando o sistema Git. Ele permite que desenvolvedores armazenem, colaborem e controlem revisões de projetos de software, facilitando a colaboração remota, o rastreamento de alterações e a integração contínua em desenvolvimento de software. Saiba mais

JQuery

O jQuery é uma biblioteca JavaScript que simplifica a manipulação de documentos HTML, o tratamento de eventos, a animação e a realização de requisições AJAX, tornando o desenvolvimento web mais eficiente e compatível entre navegadores. Saiba mais
